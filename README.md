<h2>Reverse Multidimensional Array

_Reverse an array of arbitrary elements in place along with any arrays (or arrays of arrays)._

<h3>Example:</h3> 
<p>Input: [1, 2, [3, 4, 5], [6, [7, 8], 9]]</p>
<p>Output: [[9, [8, 7],6], [5, 4, 3], 2, 1]</p>
